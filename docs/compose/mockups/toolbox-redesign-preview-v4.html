Arthas 文档: jad

jad - 反编译已运行代码

反编译指定类的源代码。可以用来查看正在运行的 Java 类的源码。

用法

jad [option] class-pattern [filter条件]

示例

# 反编译单个类
jad com.example.MyClass
# 反编译并输出到文件
jad -d /tmp arthas com.example.MyClass
# 只反编译指定方法
jad com.example.MyClass *testMethod*
