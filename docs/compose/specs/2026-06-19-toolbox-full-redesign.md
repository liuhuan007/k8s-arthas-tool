# 工具箱 UI 全面重设计

> **日期**: 2026-06-19
> **状态**: 设计完成
> **范围**: 工具箱整体布局、卡片设计、交互流程全面优化

---

## [S1] 问题定义

### 现状问题

1. **布局问题**
   - 三个区块（二进制/脚本/快捷）平铺，占据大量垂直空间
   - 统计栏信息密度低，占用空间但价值不大
   - 卡片网格 `minmax(320px, 1fr)` 在小屏幕上显示不理想

2. **卡片设计问题**
   - 二进制工具卡片包含：图标、名称、状态、类型、版本、SHA、路径、操作按钮
   - 信息过载，关键信息不突出
   - 操作按钮（校验/分发/删除）平铺，视觉权重相同

3. **交互问题**
   - 分发使用内联表单，空间受限
   - 批量分发入口不明显
   - 无搜索/过滤功能

4. **视觉问题**
   - 卡片样式偏暗，对比度不足
   - 图标颜色区分度不够
   - 空状态设计简陋

### 设计目标

- **信息层次清晰** — 关键信息突出，次要信息折叠
- **操作高效** — 常用操作一键可达，减少点击
- **视觉现代** — 符合生产级 UI 标准
- **响应式适配** — 支持不同屏幕尺寸

## [S3] 卡片设计重做

### 二进制工具卡片（新）

```
┌─────────────────────────────────────┐
│ 📦 arthas-boot.jar           可用  │
│ v3.7.2 · 内置                       │
│ /app/arthas/arthas-boot.jar         │
│ ─────────────────────────────────── │
│ [校验]  [分发→]  [删除]            │
└─────────────────────────────────────┘
```

**设计要点**：
- 名称 + 状态标签在标题行
- 版本和来源在第二行
- 路径在第三行（灰色小字）
- 操作按钮在底部，主要操作（分发）用蓝色高亮

### 脚本工具卡片（新）

```
┌─────────────────────────────────────┐
│ 🐍 CPU 分析脚本               低风险│
│ Python · 关联: CPU Profiling        │
│ ─────────────────────────────────── │
│ [编辑]  [执行→]  [删除]            │
└─────────────────────────────────────┘
```

**设计要点**：
- 运行时和风险等级在标题行
- 关联能力在第二行
- 操作按钮：编辑（次要）、执行（主要）、删除（危险）

### 快捷操作卡片（新）

```
┌─────────────────────────────────────┐
│ ⚡ jad 反编译                低风险 │
│ JVM 基础                            │
│ jad {class_name}                    │
│ ─────────────────────────────────── │
│ [执行→]  [文档]                     │
└─────────────────────────────────────┘
```

**设计要点**：
- 分类和风险等级在标题行
- 命令模板预览在中间
- 操作按钮：执行（主要）、文档（次要）

## [S4] 分发交互重做

### Modal 弹窗设计

```
┌─────────────────────────────────────────────────────────────┐
│  分发工具: arthas-boot.jar                          [✕]    │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌─ 快速选择 ─────────────────────────────────────────────┐ │
│  │ ⭐ 最近使用                                            │ │
│  │ ┌─────────────────────────────────────────────────┐   │ │
│  │ │ prod / default / pod-a-7cc5f (app)      [选择] │   │ │
│  │ │ prod / default / pod-b-8d2e1 (app)      [选择] │   │ │
│  │ └─────────────────────────────────────────────────┘   │ │
│  └────────────────────────────────────────────────────────┘ │
│                                                             │
│  ── 或手动选择目标 ──────────────────────────────────────── │
│                                                             │
│  集群:     [prod ▾]                                         │
│  Namespace:[default ▾]                                      │
│  Pod:      [pod-a-7cc5f ▾]  ✅ Java 11 + Arthas            │
│  容器:     [app ▾]                                         │
│  安装路径: [/app/arthas/arthas-boot.jar]                    │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│                                       [取消]  [确认分发]    │
└─────────────────────────────────────────────────────────────┘
```

### 交互流程

1. 点击卡片上的 **[分发→]** 按钮
2. 打开 Modal 弹窗
3. **快速选择**（可选）：点击最近使用的目标，一键填充
4. **手动选择**（如果需要）：级联选择集群/Namespace/Pod
5. 选择 Pod 后显示**能力状态**（✅/⚠️/❌）
6. 点击 **[确认分发]** → 显示进度 → 完成后关闭

## [S5] 搜索和过滤

### 搜索功能

在顶部工具栏添加搜索框：

```
[🔍 搜索工具...]
```

**行为**：
- 实时过滤（输入时立即生效）
- 搜索范围：工具名称、类型、路径
- 高亮匹配文本
- 无结果时显示空状态

### 过滤标签

在搜索框旁添加过滤标签：

```
[全部] [二进制] [脚本] [快捷]
```

**行为**：
- 点击标签切换显示的工具类型
- 默认显示全部
- 选中标签高亮
